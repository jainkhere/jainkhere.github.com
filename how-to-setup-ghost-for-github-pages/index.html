How to setup Ghost for Github Pages

25 September 2015

GitHub provides a good option for hosting static blog as GitHub Pages and there are numerous platform to set up that static blog. Many of my friends use Jekyll for that purpose but I wanted to try something new so I thought about giving Ghost a try.

Ghost is really powerful blogging platform that is also open source, what else do you need?
Yes you are right, A small tutorial so that you can too set up Ghost for your GitHub Pages.

Install Node.js

Since Ghost is based on Node.js you will need it.

#dnf install nodejs

If you are on Linux almost all chances are that you have wget, but if you don't have wget just use command

#dnf install wget

and you are good to go.

Download and install Ghost

After you have install node.js next up is downloading and installing the Ghost package.

Here I am creating an all new directory for Ghost but it all depends on you where you want to keep your files.

mkdir Ghost  
cd Ghost  
wget --no-check-certificate https://ghost.org/zip/ghost-0.7.0.zip  
unzip ghost-0.7.0.zip

These commands will download the Ghost package from The Ghost Website. Now to install it just run

npm install --production

and congratulations, you have successfully installed Ghost on your system. To start Ghost run

npm start

and Ghost is up and running locally on your system. Open http://localhost:2368 create a ghost account for yourself and read the initial blog on how to use markdown.

Create your GitHub Repo

First read official GitHub Pages documentation here and proceed to create your repo.
Do remember that your repo name should be of form username.github.io and username should be in lower case.

Export Ghost content using Buster

Once you have installed Ghost and also created repo, it is now turn to export your Ghost content to your GitHub Pages and start your static blog. Buster is really good tool to do so without any hassle. Install Buster with:

#dnf install pip
#pip install buster

After installation is Complete, simply run following commands to export your content

buster setup --gh-repo=<your repo url>  
buster generate --domain=http://localhost:2368  
buster deploy

With this Buster adds, commits and pushes your Ghost content to your GitHub repo. All you have to do now is open http://username.github.io and admire the beauty of Ghost on your blog.

For writing and exporting new posts just start ghost and then deploy content using buster.

That's it and Happy Blogging : )
